## **Proyecto DAM.**  
### **Second Skin**  
### **Descripción:**  
Aplicación multiplatforma desarrollada en Dart para que sea lo mas nativo posible a Swift y a Kotlin.  
### **Estructura de la APP**  
  
La aplicación se compone de un home, un menu lateral y los productos
